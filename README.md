# PoiskFlimsLab34 - Лабораторная работа 3-4

## Приложение для поиска фильмов

Angular приложение для поиска фильмов с использованием OMDB API с системой избранного, кэшированием и unit-тестированием.

## Требования (ТЗ)

### Лабораторная 3:
✅ Поиск фильмов через OMDB API  
✅ Использование RxJS (debounceTime, switchMap, catchError, retry)  
✅ Типизированные интерфейсы для API  
✅ Standalone компоненты  
✅ Карточки фильмов с постером, годом, названием  

### Лабораторная 4:
✅ Детальная информация о фильмах  
✅ Фильтры по типу (фильмы/сериалы)  
✅ Сервис избранного с localStorage  
✅ Сервис кэширования для оффлайн-режима  
✅ Unit-тестирование сервисов  

##  Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск приложения
ng serve

# Запуск тестов
ng test

# Покрытие кода
